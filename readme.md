# Tera-Guide

是一个通过简单的“脚本”非常容易地创建副本補助的模块。

# #依赖性

https://github.com/michengs/library

需要libray模组依赖 下载解压放入 proxy/mods文件夹目录


## 命令

默认系统通知开启！！！

proxy频道输入:

補助
- 模组开/关
- 默认: On


補助 语音
- 语音通知开/关
- 默认: On



補助 组队长通知
-组队长通知开/关
- 默认: off


補助 组队通知
-组队通知开/关
- 默认: off




補助 通知
- 通知开/关
- 默认: On



補助 2~10

- 调节语音速度10为最快语速，默认为正常语速1







## 语音依赖

https://github.com/michengs/voice

下载并解压到 proxy/node_modules里面注意重命名为:voice
