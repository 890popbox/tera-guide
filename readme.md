# Tera-Guide

是一个通过简单的“脚本”非常容易地创建副本補助的模块。

# #依赖性
https://github.com/michengs/library




## 命令
proxy频道输入:

補助
- 模组开/关
- 默认: On


補助 语音
- 语音提示开/关
- 默认: On

補助 提示
- 提示开/关
- 默认: On
